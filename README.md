# wiStopWatch

This software will function as a stopwatch. It will include wireless capabilities in order to send the data collected into an excel sheet.

Function:
1. Record time spent on an activity
2. Record how many units processed
3. Send collected data to PC in excel form

Software used:
1. Arduino 1.8.15
2. I2C LCD Library http://www.esp8266learning.com/wp-content/uploads/2017/12/LiquidCrystal_I2C-master.zip
3. Wemos D1 Mini board driver https://arduino.esp8266.com/stable/package_esp8266com_index.json

Hardware used:
1. Wemos D1 mini board x1
2. I2C LCD x1
3. Push button module x2
